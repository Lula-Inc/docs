---
title: "Get All Order Items"
api: "GET {{micro_service_base_url}}/orders/{{order_id}}/order-items"
description: "Retrieve all items within a specific order with complete details, modifications, and fulfillment status for each item."
playground : "simple"
---

This endpoint returns a comprehensive list of all items within an order. It's ideal for order fulfillment, kitchen display systems, packing lists, and complete order review.

<Info>
This endpoint provides complete item details for all items in an order, making it efficient for bulk operations and comprehensive order management.
</Info>

### Path Parameters

<ParamField path="order_id" type="string" required>
  The unique identifier of the order whose items you want to retrieve
</ParamField>

### Query Parameters

<ParamField query="include_modifications" type="boolean" optional default="true">
  Whether to include detailed modification information for each item
</ParamField>

<ParamField query="status_filter" type="string" optional>
  Filter items by fulfillment status: "pending", "preparing", "ready", "completed"
</ParamField>

<ParamField query="category_filter" type="string" optional>
  Filter items by product category
</ParamField>

### Response

<ResponseField name="order_id" type="string">
  The order identifier these items belong to
</ResponseField>

<ResponseField name="total_items" type="integer">
  Total number of items in the order
</ResponseField>

<ResponseField name="items" type="array">
  Array of all order items with complete details

  <Expandable title="Order Item Object">
    <ResponseField name="id" type="string">
      Unique order item identifier
    </ResponseField>

    <ResponseField name="quantity" type="integer">
      Quantity of this item ordered
    </ResponseField>

    <ResponseField name="special_instructions" type="string">
      Item-specific preparation instructions
    </ResponseField>

    <ResponseField name="store_item_price" type="string">
      Price per unit for this item
    </ResponseField>

    <ResponseField name="store_item_total_price" type="string">
      Total price for this item (quantity Ã— unit price)
    </ResponseField>

    <ResponseField name="store_item_name" type="string">
      Product name as displayed in store
    </ResponseField>

    <ResponseField name="store_item_description" type="string">
      Product description
    </ResponseField>

    <ResponseField name="store_item_category" type="string">
      Product category classification
    </ResponseField>

    <ResponseField name="fulfillment_status" type="string">
      Current fulfillment status for this item
    </ResponseField>

    <ResponseField name="item" type="object">
      Complete product information including images, UPC, and categories
    </ResponseField>

    <ResponseField name="modifications" type="array">
      Applied modifications/customizations (when include_modifications=true)
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="order_summary" type="object">
  Summary information for the complete order

  <Expandable title="Order Summary">
    <ResponseField name="total_quantity" type="integer">
      Total quantity of all items
    </ResponseField>

    <ResponseField name="subtotal" type="string">
      Subtotal of all items before taxes
    </ResponseField>

    <ResponseField name="items_by_category" type="object">
      Count of items grouped by category
    </ResponseField>

    <ResponseField name="items_by_status" type="object">
      Count of items grouped by fulfillment status
    </ResponseField>
  </Expandable>
</ResponseField>

<Accordion title="Filter Examples">

**Get Items by Status**
```
GET {{micro_service_base_url}}/orders/{{order_id}}/order-items?status_filter=pending
```

**Get Items by Category**
```
GET {{micro_service_base_url}}/orders/{{order_id}}/order-items?category_filter=Baby
```

**Get Items Without Modifications**
```
GET {{micro_service_base_url}}/orders/{{order_id}}/order-items?include_modifications=false
```

</Accordion>

<Note>
**Performance Optimization**: When you only need basic item information, set include_modifications=false to reduce response size and improve performance.
</Note>

<Tip>
**Kitchen Display**: Use status_filter=pending to show only items that need preparation, perfect for kitchen display systems.
</Tip>
