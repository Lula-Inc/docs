## JSON Object Reference

Here are the accepted and required JSON fields for inventory items:

### Fields

<ResponseField name="external_id" type="string" required>
  ID from your source system, used to update existing items.
</ResponseField>

<ResponseField name="quantity" type="number" required>
  Quantity of the product in inventory.
</ResponseField>

<ResponseField name="status" type="string">
  Inventory status (optional).
</ResponseField>

<ResponseField name="price" type="number" required>
  Product price.
</ResponseField>

<ResponseField name="location" type="string">
  Product location (optional).
</ResponseField>

<ResponseField name="product_id" type="number">
  Internal product ID (optional).
</ResponseField>

### Example JSON

```json Example
[
  {
    "external_id": "1234_5678",
    "quantity": 99,
    "status": "active",
    "price": 265,
    "location": "Aisle 3",
    "product_id": 101
  },
  {
    "external_id": "9876_5432",
    "quantity": 50,
    "price": 250
  }
]
```
